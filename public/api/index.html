À propos de ce Pipeline

Ce pipeline DevSecOps automatise le déploiement sécurisé de l'application Akaunting sur AWS EKS. Chaque commit déclenche une série d'étapes de sécurité, de tests et de déploiement.

Fonctionnalités principales:

  • Scan de sécurité automatique avec Trivy
  • Build et push d'images Docker
  • Déploiement sur Kubernetes (AWS EKS)
  • Tests automatisés et validation
  • Monitoring et alertes en temps réel
